# Icons

Extension cần 3 file icon:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

## Tạo Icons

Bạn có thể tạo icons bằng cách:

1. Sử dụng tool online như [Favicon Generator](https://www.favicon-generator.org/)
2. Sử dụng image editor (Photoshop, GIMP, etc.)
3. Sử dụng online icon maker

## Yêu cầu

- Format: PNG
- Kích thước: 16x16, 48x48, 128x128 pixels
- Nền trong suốt (transparent background) được khuyến nghị
- Màu chủ đạo: #1a73e8 (Google Blue)

## Placeholder

1. Tạo hình vuông màu #1a73e8
2. Thêm text "AF" (Auto Flow) màu trắng ở giữa
